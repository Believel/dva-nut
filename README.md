# usage
## create a dva project
```js
// 1. install dva-cli
npm install dva-cli -g
dva -v
// dva-cli version 0.10.1

// 2. create new app
dva new dva-quickstart

// 3. run app
cd dva-quickstart
npm start

// 4. build app 
npm run build
// generate dist/
```
